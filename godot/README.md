# Gribbits Town - Godot Web Version

## Come Testare Localmente

1. Apri la cartella `godot/` con Godot 4.3+
2. Apri la scena `MainWorld.tscn`
3. Premi Play

## Controlli
- **WASD** → Muoviti
- **Mouse** → Guarda intorno
- **Spazio** → Salta
- **Click Sinistro** → Piazza blocco
- **Click Destro** → Rimuovi blocco
