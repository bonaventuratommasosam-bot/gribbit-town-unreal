## Controlli
- **WASD** → Muoviti
- **Mouse** → Guarda intorno
- **Spazio** → Salta
- **Click Sinistro** → Piazza blocco
- **Click Destro** → Rimuovi blocco

## Come Provare

1. Apri la cartella `godot` con Godot 4.3+
2. Apri `MainWorld.tscn`
3. Premi Play
